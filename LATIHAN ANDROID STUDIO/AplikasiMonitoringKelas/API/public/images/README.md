# Logo SMENDA

## Instruksi Penggunaan:

1. Letakkan file logo SMENDA dengan nama `logosmenda.png` di folder ini
2. Format yang disarankan: PNG dengan background transparan
3. Ukuran yang disarankan: 512x512 pixels atau lebih tinggi
4. Jika logo tidak tersedia, sistem akan menggunakan icon default (graduation cap)

## Path Logo:

```
/public/images/logosmenda.png
```

## Lokasi Penggunaan:

-   Halaman Login (`/login`)
-   Sidebar Dashboard (Semua halaman admin)

## Format Alternative:

Jika Anda ingin menggunakan format lain (JPG, SVG, dll), ubah ekstensi di:

-   `resources/views/auth/login.blade.php`
-   `resources/views/layouts/admin.blade.php`
